Modelica.Electrical.Digital.Interfaces.Logic

Signal type in package Digital according to the IEEE 1164 STD_ULOGIC type


Information






A variable of type Logic is an Integer in the range 1-9.
The Integer values have the following meaning:





  
Logic value
      Integer code
      Meaning
  


  
'U' 1 Uninitialized

  
'X' 2 Forcing Unknown

  
'0' 3 Forcing 0

  
'1' 4 Forcing 1

  
'Z' 5 High Impedance

  
'W' 6 Weak Unknown

  
'L' 7 Weak 0

  
'H' 8 Weak 1

  
'-' 9 Don't care
