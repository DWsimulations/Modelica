Information

Version 3.2.1 build 3 is backward compatible to version 3.2.1 build 2, that is models developed with versions 3.0, 3.0.1, 3.1, 3.2 or 3.2.1 will work without any changes also with version 3.2.1 build 3. This version is a bug-fix release. Short Overview:

  • This version of the Modelica package is fully compatible to Modelica Specification 3.2 revision 2.
  • About 103 tickets have been fixed for this release.
  • No models and blocks and functions are newly included.
