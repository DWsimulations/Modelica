.ModelicaReference.Operators.'cosh()'

Information

Hyperbolic cosine function

Syntax

cosh(u)

Description

Returns the cosh of u, with -∞ < u < ∞. Argument u needs to be an Integer or Real expression.

The cosh function can also be accessed as Modelica.Math.cosh.

Examples

cosh(1)
  = 1.54308063481524
