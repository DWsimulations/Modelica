.ModelicaReference.Operators.'sqrt()'

Information

Square root

Syntax

sqrt(v)

Description

Returns the square root of v if v>=0, otherwise an error occurs. Argument v needs to be an Integer or Real expression.

Examples

sqrt(9)
 = 3.0
